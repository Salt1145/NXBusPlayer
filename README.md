# NXBusGPS Announcer - 公交GPS报站模拟器

> 一个模拟真实公交GPS自动报站系统的Web应用，支持多种数据格式、GPS触发、手动控制等功能。

## 简介

NXBusGPS Announcer 是一个基于浏览器的公交报站模拟器，能够读取预设的线路数据，结合GPS位置信息自动触发进站/出站语音播报和LED文字显示。它支持多种公交报站数据格式（APEP、博闻、中安、凯伦等），并提供完整的灵敏度调节、方向切换、手动跳站等功能。
*本项目为纯ai编写，且实现方式为纯静态

## 主要功能

- **GPS自动触发**：根据当前位置与站点距离，自动播放进站/出站语音
- **手动控制**：支持下一站、重放、切换方向、打断音频等操作
- **多种格式支持**：APEP、APEP TypeB、博闻（含/无GPS）、中安、凯伦、卡密、CSV后备
- **线路扫描**：自动扫描`linelist.txt`中的线路，支持缓存加速
- **灵敏度调节**：可分别设置出站/进站触发距离，支持预设方案
- **LED滚动显示**：模拟真实公交车的红色LED滚动文字（中英文双语）
- **站点管理**：点击站点跳转、删除站点、颠倒上下行
- **密码保护**：支持对特定线路设置访问密码（`passwd.wd`）
- **坐标转换**：支持WGS-84与GCJ-02坐标系切换

## 使用说明

### 1. 首次启动
- 打开`index.html`，系统会自动读取`linelist.txt`列出所有线路（初始状态为“未扫描”）
- 点击“扫描当前”或“全量扫描”按钮，系统会加载对应线路的音频配置和站点信息
- 扫描完成后，线路会显示格式标记（如`[博闻]`、`[中安]`等）和GPS状态

### 2. GPS自动模式
- 允许浏览器定位权限后，系统会实时计算当前位置与各站点的距离
- 当车辆进入站点半径（可调灵敏度）时，自动播放进站语音和LED文字
- 驶离站点半径时，自动播放出站语音（下一站预告）

### 3. 手动模式
- 使用“下一站”按钮：在当前状态（进站）时触发下一站预告，出站时触发进站
- “重放”：重新播放当前站点（进站）或上一站出站（下一站）语音
- “切换方向”：切换上下行线路（若存在）
- “打断音频”：立即停止当前所有音频播放

### 4. 站点管理
- 点击站点列表中的站点：立即跳转至该站点（作为进站状态）
- 点击“×”按钮：删除该站点（不可逆）
- 点击“颠倒”按钮：交换上下行线路的所有站点数据

### 5. 灵敏度调节
- 通用灵敏度：范围触发模式（APEP/中安等）时，站点判断半径
- 出站/进站灵敏度：博闻格式（点触发）时，分别匹配出口/入口点半径

## 支持的音频格式

| 格式类型   | 识别方式                      | 音频文件示例                |
| ---------- | ----------------------------- | --------------------------- |
| APEP       | 存在`station.json`            | `a001_up.mp3` / `l001_up.mp3` |
| APEP TypeB | 存在`stslist.json`            | `001au.mp3` / `001bu.mp3`     |
| 博闻（有GPS） | 存在`gps.txt`且无`station.json` | `001au.mp3` / `001ad.mp3`     |
| 博闻（无GPS） | 存在`au/ad`无`gps.txt`且无`cu` | 同上                        |
| 中安        | 存在`001cu.mp3` + CSV匹配     | `001cu.mp3` / `001au+001bu`   |
| 凯伦        | 存在`001au-002-Up-Start.mp3`  | 长文件名格式                |
| CSV后备    | 其他格式均不匹配，但有CSV数据 | 按APEP命名规则              |

## 配置示例

### `linelist.txt` 格式说明
```
# 普通线路
浦东21路

# 禁用CSV（*开头）
*浦东21路

# 指定上下行音频数量（((上行数,下行数))）
浦东21路((20,18))

# 过滤站点（<站点名1,站点名2>）
浦东21路<招呼站,临时站>

# 交换上下行（末尾加?）
浦东21路((20,18))?
```

### `passwd.wd` 格式
```
浦东21路:123456
```

### `notice.tz` 格式
支持纯文本，可以使用`[文件名]`引用图片或视频：
```
欢迎使用NXBusGPS模拟器！
当前版本：V1.7
[images/bus.jpg]
```

## 注意事项
- 本应用需要**HTTPS**或**localhost**环境才能获取高精度GPS（非安全上下文仅允许低精度定位）
- 音频文件必须与配置文件命名完全一致，区分大小写
- CSV数据中的站名若包含`·`，其后内容将作为备注显示（不影响匹配）
- 缓存存储在浏览器的`localStorage`中，如需重新扫描所有线路请点击“全量扫描”
